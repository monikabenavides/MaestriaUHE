# 🚀 Perfil Epidemiológico y Factores de Riesgo en Pacientes con Infecciones Respiratorias Agudas Graves (IRAG)

Este proyecto está enfocado en la vigilancia epidemiológica y el análisis estadístico de las Infecciones Respiratorias Agudas Graves (IRAG) en el Ecuador, tomando como base el histórico de pacientes con cuadros clínicos severos cuyo desenlace fue el fallecimiento.

El objetivo principal es identificar patrones demográficos, temporales y clínicos que permitan entender mejor el comportamiento y el impacto de estas enfermedades en el país.
